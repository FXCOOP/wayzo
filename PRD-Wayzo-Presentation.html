Wayzo — PRD דו‑לשוני (HE+EN) — Product Requirements Document

📘 PRD — עברית (ממוקד אופטימיזציה + SEO)

תקציר מנהלים

Wayzo הוא עוזר אישי חכם לטיולים: יצירת תוכניות ברות‑ביצוע (כבר עובד: טופס → AI → PDF/ICS → ווידג'טים להזמנה), ובהמשך ליווי בזמן אמת (התראות, שינוי מסלול לפי מזג אוויר/עומסים). מודל הכנסה: תשלום פר‑תוכנית + Upsell שנתי, עם אפיליאייט כעמוד מרכזי.

יעדים & היקף

  • תוכניות איכותיות וברורות: תקציר, מסלול יומי, לינה, מסעדות, תחבורה, תקציב מדויק.
  • טיפוסים כלליים תחילה: משפחותאנשי עסקיםיחידים ספונטניים (טיולים יומיים).
  • ליווי בזמן אמת בהטמעה הדרגתית: התראות, "עדכן מסלול" דינמי.
  • SEO כתעלת צמיחה מרכזית (i18n, JSON‑LD, CWV).

טכנולוגיה — "Newer Tech" + SEO מושלם

  • Frontend: Next.js 15 (App Router) + React Server Components + TypeScript + Tailwind; RTL/LTR.
  • Rendering/SEO: SSR/SSG/ISR; i18n routing + hreflang; canonical; OpenGraph/Twitter; XML sitemap; robots.txt; preload/preconnect; Core Web Vitals ≥95.
  • Schema.org: JSON‑LD ל־Itinerary/TouristTrip/Hotel/Restaurant/Event לתצוגות עשירות.
  • Backend: Node/Express TS; קצוות Edge ל‑preview/plan; ולידציות קשיחות; rate‑limit.
  • Data: Postgres + Prisma; Redis כ‑cache ל‑AI/מזג אוויר/מפות עם TTL אדפטיבי.
  • AI Orchestration: תבניות prompt + guardrails (regex/JSON validators), cost caps, link sourcing ל‑widgets, auto‑validation + scoring.
  • Observability: pino logs, OpenTelemetry tracing, metrics + התראות.
  • Security/Perf: Helmet/CSP/CORS; HTTP/3; CDN; AVIF/SVG; SLOs.

🌍 i18n & SEO בינלאומי (שפות: EN, IT, FR, DE, ES, RU, HE, JA)

  • מבנה URL: /{lang}/... (לדוגמה: /en/, /it/). x-default לנתיב הראשי.
  • hreflang: הזרקה אוטומטית לכל עמוד; ניהול canonical למניעת כפילויות.
  • Schema.org JSON‑LD: כולל @language היכן שרלוונטי (Itinerary, TouristTrip, Hotel, Restaurant, Event).
  • תרגום/לוקליזציה: next‑intl + TMS (Phrase/Lokalise). מילונים לפי תחביר ICU; בדיקת LQA בסיסית.
  • תוכן SEO: Hubs לפי יעד × סוג טיול × עונה; ISR לתסריטי Long‑Tail; מחקר מילות מפתח per‑locale.
  • פורמט/מטבע: פורמט תאריך/שעה מקומי; המרת מטבע; PDF/ICS לוקאלי; RTL רק לעברית.
  • פונטים: Inter (Latin), Noto Sans Hebrew, Noto Sans JP; fallbacks מלאים ל‑Cyrillic/CJK.

📘 PRD — English (Optimization + SEO‑First)

Executive Summary

Wayzo is an AI travel companion: generate bookable plans (already live: form → AI → PDF/ICS → booking widgets) and evolve to real‑time assistance (alerts, weather/traffic‑aware rerouting). Monetization: pay‑per‑plan + attractive annual upsell; affiliates as a core revenue stream.

Goals & Scope

  • High‑quality, bookable itineraries: overview, day‑by‑day, hotels, dining, transport, precise budget.
  • General trip types first: familiesbusinesssolo/spontaneous day trips.
  • Real‑time companion via incremental rollout: alerts, "Update route".
  • SEO as a primary growth engine (i18n, JSON‑LD, CWV).

Tech — Newer Stack & SEO‑Perfect

  • Frontend: Next.js 15 (App Router), React Server Components, TypeScript, Tailwind; RTL/LTR.
  • Rendering/SEO: SSR/SSG/ISR; i18n routing + hreflang; canonical; OpenGraph/Twitter; XML sitemap; robots.txt; preload/preconnect; Core Web Vitals ≥95.
  • Schema.org: JSON‑LD for Itinerary/TouristTrip/Hotel/Restaurant/Event rich results.
  • Backend: Node/Express TS; Edge‑ready endpoints for preview/plan; strict validators; rate‑limits.
  • Data: Postgres + Prisma; Redis cache for AI/weather/maps with adaptive TTL.
  • AI Orchestration: templated prompts + guardrails (regex/JSON validators), cost caps, deterministic link sourcing to widgets, auto‑validation + scoring.
  • Observability: Pino logs, OpenTelemetry tracing, metrics + alerts.
  • Security/Perf: Helmet/CSP/CORS; HTTP/3; CDN; AVIF/SVG; SLOs.

🌍 International i18n & SEO (Languages: EN, IT, FR, DE, ES, RU, HE, JA)

  • URL structure: /{lang}/... (e.g., /en/, /de/); add x-default.
  • hreflang: automatic injection per page; strict canonical handling.
  • Schema.org JSON‑LD: include @language where applicable.
  • Localization pipeline: next‑intl + TMS (Phrase/Lokalise); ICU messages; basic LQA checks.
  • Content hubs: destination × trip type × season; ISR for long‑tail; locale keyword research.
  • Formats/Currency: locale date/time; currency conversion; localized PDF/ICS; RTL only for HE.
  • Fonts: Inter (Latin), Noto Sans Hebrew, Noto Sans JP; complete fallbacks for Cyrillic/CJK.
  • Language → hreflang map: en, it, fr, de, es, ru, he, ja + x‑default.
